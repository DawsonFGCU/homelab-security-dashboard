About this system

I installed Linux on an old Lenovo ThinkPad and built this server to learn how real infrastructure works and to demonstrate my skills. Every box in the diagram is a real, working part of it — here's the role each one plays:

  • Public visitor — the public entry point; anyone can open the live demo and interact with it.
  • Private admin — that's me, connecting as administrator to build, configure, and maintain the server.
  • Cloudflare Tunnel — serves the public demo over HTTPS without opening a single port on my home network.
  • Tailscale VPN — gives me encrypted private access (WireGuard) to administer the server, with nothing exposed to the internet.
  • Nginx reverse proxy — the single front door: it receives every request and passes it inward, so the server is never directly exposed.
  • FastAPI backend — the Python app behind the dashboard; it stays bound to localhost so only Nginx can reach it.
  • psutil — reads live CPU, memory, disk, and network stats straight from the operating system.
  • Security logs — surface failed SSH logins and auto-ban repeat offenders (journald + fail2ban) for live intrusion detection.
  • Ollama — runs a local AI model on the machine, so the assistant answers questions with no data ever leaving the server.
